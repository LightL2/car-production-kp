# Image & Video Production — КП · 8BIT-MEDIA

Интерактивная HTML-презентация коммерческого предложения по фото/видео продакшну.

## Открыть

| Файл | Слайды | Назначение |
|------|--------|------------|
| [`index.html`](index.html) | 25 | **Главная на хостинге** — полная версия для клиента |
| [`presentation.html`](presentation.html) | 25 | То же (альтернативный URL) |
| [`kp.html`](kp.html) | 16 | Исходник: только КП (без блока команды) |

**Навигация:** ← → / пробел · **F** — полный экран · **P** — PDF · свайп на телефоне

На мобильных презентация открывается масштабом как на ПК (1280px), с pinch-to-zoom.

## Видео BYD / DENZA (слайд «Знаем продукт бренда»)

Загружайте на хостинг в папку:

```
car-production-kp/assets/byd/
```

| Файл | Назначение |
|------|------------|
| `denza-35.mp4` | DENZA · вертикальный reel (верхний ряд, слева) |
| `byd-cinematic.mp4` | BYD · cinematic · вертикальный reel (верхний ряд, справа) |
| `byd-4x3-final.mp4` | BYD · 4×3 · горизонтальный ролик (нижний ряд) |

## Пересборка

```powershell
cd car-production-kp
python build_full_presentation.py
```
